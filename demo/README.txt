To run demo.ipynb, you must install
1) if using Anaconda3:
- conda create -n p36 python=3.6.8
- conda activate p36
- conda install seaborn
- pip install tensorflow

2) If not using Annoconda:
- install python 3.6.*
- pip3 install tensorflow
- pip3 install seaborn
- pip3 install matplotlib

See the mp4 file to get details of the demo.ipynb file
